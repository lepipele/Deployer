Copy deployer-full.xsd and deployer-intellisense.xml to your Visual Studio’s xml\schemas path. 

Default path for VS2005: %ProgramFiles%\Microsoft Visual Studio 8\Xml\Schemas\ 
Default path for VS2008: %ProgramFiles%\Microsoft Visual Studio 9.0\Xml\Schemas\
Default path for VS2010: %ProgramFiles%\Microsoft Visual Studio 10.0\Xml\Schemas\
Default path for VS2012: %ProgramFiles%\Microsoft Visual Studio 11.0\Xml\Schemas\

Restart Visual Studio and when you open it again and create New XML file, you'll get deployer intellisense
